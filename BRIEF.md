Riferimento file: [assets/menu/240418 LuxNight_Menu_4pg_105x210mm_H.pdf](assets/menu/240418%20LuxNight_Menu_4pg_105x210mm_H.pdf)

- **Logo**: lettering "LUX NIGHT" su due righe, font outline tondo/bubble (stile neon), con due stelline a 5 punte ai lati della parola "LUX".
- **Palette**:
  - Sfondo principale: viola scuro / bordeaux molto scuro (~ `#2A0A1F` / `#3B0F2A`)
  - Accento centrale (radiale, glow): magenta vibrante (~ `#A8124B` / `#C21B5A`)
  - Testo / linee: bianco (`#FFFFFF`)
- **Effetto grafico**: glow radiale al centro + cornice sottile bianca → mood "neon notturno premium".
- **Tipografia**:
  - Display (titoli/logo): sans condensata maiuscola con stile **outline / neon** (es. *Monoton*, *Bungee*, *Beon* — da confermare con originale)
  - Testo corrente: sans serif bianca, peso medio/bold per le voci di menù
- **Tono di voce**: notturno, esclusivo, glamour, leggermente provocante (in linea con i social TikTok).

> Il **logo vettoriale** (SVG/AI) **non è ancora nei materiali forniti** — chiederlo al cliente. Per ora si dispone solo del rendering dal PDF.

## 8. Struttura sito consigliata

Sezioni minime per il sito Lux Night:

1. **Home / Hero**
   - Video o slideshow ambientato (dalle foto in [assets/foto/](assets/foto/))
   - Logo + tagline *"Il night più esclusivo del Ticino"*
   - CTA: **Prenota tavolo** (tel/WhatsApp/form)
2. **Il locale** — racconto atmosfera, posizionamento, perché Lux Night
3. **Gallery** — selezione foto sala, privé, bottiglie, eventi (16 foto disponibili in `assets/foto/`)
4. **Menu / Drink list** — selezione drink, bottiglie, champagne, privé (da [content/menu.md](content/menu.md) o link al PDF)
5. **Eventi** — calendario serate, DJ, ospiti
6. **Privé** — descrizione sale Privé Rosa & Blu, modalità di prenotazione
7. **Prenotazioni** — form / WhatsApp diretto / telefono
8. **Contatti & come arrivare** — mappa Google, indirizzo, telefono, email, orari, parcheggio
9. **Footer** — social, P.IVA/ragione sociale, privacy, cookie, impressum (obbligatorio in CH)

### Note legali svizzere da prevedere
- **Impressum** (obbligatorio in Svizzera): ragione sociale, indirizzo, contatti, persona di riferimento.
- **Privacy** conforme alla **nLPD** (nuova Legge svizzera sulla Protezione dei Dati, in vigore dal 1° settembre 2023).
- **Cookie banner** se si usano analytics o pixel.
- **Disclaimer 18+** consigliato in apertura (cabaret/alcol).

### Sito web (`site/`)
- Stack: **HTML + CSS + JS vanilla**, nessun build step, nessuna dipendenza npm. Hosting compatibile con qualsiasi static host (Netlify, Vercel, GitHub Pages, hosting tradizionale).
- **One-page** con disclaimer 18+ all'apertura (memorizzato per la sessione in `sessionStorage`).
- Sezioni implementate: Hero, Il locale, Gallery (con lightbox), Menu, Privé, Eventi, Prenota, Contatti & mappa, Footer.
- Mobile-first, responsive, dark theme con palette viola/magenta + bianco.
- Tipografia: **Bungee Outline** (display, logo neon) + **Inter** (body) da Google Fonts.
- SEO base: title, description, OpenGraph, JSON-LD `NightClub` con orari e indirizzo.
- Le foto sono linkate via `../assets/foto/` (path relativi). Per il deploy basta copiare le cartelle `site/` + `assets/` mantenendo la stessa struttura.

#### Test locale
```bash
# dalla cartella LUXNIGHT/
python -m http.server 8765
# poi apri http://127.0.0.1:8765/site/
```
